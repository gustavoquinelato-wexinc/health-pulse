Role-Based Access Control

Permissions are managed through user roles:

  • Admin: Full system access to all resources and actions
  • User: Can view and use dashboards, view ETL status, download logs
  • View: Read-only access to dashboards and basic system information

Permission Levels:

  • EXECUTE includes read permissions (can perform actions)
  • READ view-only access

To change a user's permissions, edit their role in the User Management section above.
